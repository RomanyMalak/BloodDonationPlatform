# Blood Donation Backend Skeleton

This is a clean four-layer ASP.NET Core project skeleton.

Included:
- Domain layer: database entities and enums only.
- Infrastructure layer: AppDbContext only, prepared for SQL Server.
- Application layer: folders and empty placeholder classes only.
- API layer: folders and empty controller classes only, with minimal Program.cs for startup.

Not included intentionally:
- No business logic.
- No services implementation.
- No repository implementation.
- No endpoints implementation.
- No AI Agent implementation yet.

The goal is to let team members implement their parts in the correct layer without random code placement.
